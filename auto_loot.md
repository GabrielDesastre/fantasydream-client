# 🎯 Sistema de Auto Loot

Colete itens automaticamente durante suas aventuras!

## Como Funciona

- Configure quais itens coletar
- Organize seu inventário automaticamente
- Maximize sua eficiência de farm
- Sistema inteligente de priorização

## Configurações Disponíveis

- **Filtro por raridade**: Comum, Raro, Épico, Lendário
- **Filtro por tipo**: Armas, Armaduras, Consumíveis, Materiais
- **Valor mínimo**: Defina o valor mínimo dos itens
- **Capacidade**: Respeita o limite de peso do personagem

## Dicas

- Use filtros para otimizar seu farm
- Configure diferentes perfis para cada área
- Ative notificações para itens raros
- Revise periodicamente suas configurações

**Aproveite ao máximo o sistema de Auto Loot!**
